### Title: refactor(core): Eradicate Admin Dashboard Ecosystem & Reinforce Static Data Layer

### Narrative Synthesis

*Global impact on the Cyber-Premium vision, architectural decisions, and value delivered.*

This structural update significantly slims down the application by removing the legacy and partially implemented Admin Dashboard ecosystem. By eliminating the heavy zero-trust API protection, specialized routing, and CMS dependencies, the overall architecture is simplified, faster, and tightly aligned with a robust static data and headless MDX content management approach. The build process was also fortified by introducing missing core type and icon dependencies, resulting in a cleaner, zero-warning production build.

### Granular Modifications

- **[Core Routing & API]**: 
  - Entirely removed the `src/app/api/admin/` ecosystem alongside its dynamic routing endpoints.
  - Stripped out the zero-trust admin authentication, session verification, and associated rate-limiting logic from the Edge middleware (`src/proxy.ts`).
- **[UI Components & State]**: 
  - Deleted the complete `src/components/admin/` directory, purging the extensive dashboard UI library built on shadcn/ui.
  - Eliminated admin-specific Zustand stores (`src/store/admin/`), custom hooks (`src/hooks/admin/`), and utilities (`src/lib/admin/`).
- **[Internationalization (i18n)]**: 
  - Purged 5 dedicated admin dictionaries (`admin.*.json`).
  - Simplified `src/lib/dictionary.ts` and `src/types/locale.ts` by removing all traces of `AdminDictionary` and `getAdminDictionary`.
- **[Data Decoupling]**: 
  - Replaced dynamic CMS data fetching (`listContent`, `getContent`) in the Home page (`src/app/[lang]/(web)/page.tsx`) and the OpenGraph generator (`src/app/api/og/route.tsx`) with statically imported local typescript payloads (`gallery-data.ts` and `personal.ts`).
- **[Build & Types]**: 
  - Installed `@types/mdx` to correct `next-mdx-remote` compilation failures.
  - Installed `@radix-ui/react-icons` to resolve missing Radix UI imports within the Bento Grid component.

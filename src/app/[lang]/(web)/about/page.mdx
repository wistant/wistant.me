# About

I'm a software engineer specialized in forging high-end digital solutions.
I'm originally from Douala, Cameroon. I owe much of my career to the Web
and the global Open Source community.

I spent my early years advocating for high-performance software and
teaching regional developers how to leverage distributed systems. Later,
I developed a deep passion for JavaScript ecosystems and cross-platform
architecture.

import { getPageMetadata } from "@/config/metadata";

export async function generateMetadata(props) {
  const params = await props.params;
  const { lang } = params;
  return getPageMetadata(lang, {
    title: "AWS Certified Solutions Architect – Associate",
    description: "Certification: AWS Certified Solutions Architect – Associate from Amazon Web Services (AWS)",
  });
}

La certification **AWS Certified Solutions Architect – Associate** (SAA-C03) valide ma capacité à concevoir et à déployer des systèmes sécurisés et robustes sur les technologies AWS.

<Callout type="info">
Cette certification est le standard de l'industrie pour démontrer une maîtrise de l'architecture cloud bien architecturée (Well-Architected Framework).
</Callout>

### Domaines d'Expertise
- **Architectures résilientes** : Conception de systèmes hautement disponibles et tolérants aux pannes.
- **Architectures performantes** : Optimisation du calcul, du stockage et de la mise en réseau.
- **Sécurité logicielle** : Mise en œuvre du principe du moindre privilège et du chiffrement des données.
- **Optimisation des coûts** : Sélection des services et types d'instances les plus rentables.

# Admin Route Group `(admin)`

This directory contains the Content Management System (CMS) interface.

## Structure

- **`layout.tsx`**: A clean, full-screen layout specifically for the Admin UI (no website navigation).
- **`keystatic/`**: The Keystatic dashboard entry point.

## Access

- **Local:** Accessible at `/keystatic`.
- **Production:** Accessible at `/keystatic` (protected by GitHub OAuth if configured).

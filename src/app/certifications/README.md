# Certifications Module

Displays professional certifications, diplomas, and badges.

## Directory Structure

| Directory/File | Description |
| :--- | :--- |
| `content/` | **Data Source.** `.mdx` files for each certification. |
| `components/` | **Local Components.** (e.g., `BadgeDisplay`, `VerifyButton`). |
| `page.tsx` | List view of all certifications. |

## Notes

This module is primarily static but uses Content Collections to ensure type safety for certification dates and issuers.

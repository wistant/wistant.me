# Elysian Drive : Location de Voitures de Luxe 🏎️

Elysian Drive est une plateforme de location de voitures de luxe. Elle implémente des choix esthétiques sophistiqués pour résonner avec une clientèle haut de gamme cherchant l'exclusivité.

## L'Expérience Elysian

![Page d'Accueil Elysian Drive](/portfolio/elysindrive.png)

L'application utilise des images haute fidélité et une typographie audacieuse et confiante pour transmettre un sentiment d'élégance et de performances robustes.

## Vitrine Responsive

![Véhicules Elysian Drive](/portfolio/elsyandrive-wistant.netlify.app.webp)

Construit sur React avec une architecture HTML/CSS, le frontend assure une large compatibilité sur tous les appareils tout en conservant l'aspect luxueux d'une application native.

# Splyt: High-Performance Financial Dashboard 💸

Splyt is designed to handle complex algorithmic data and present it through a clean, intuitive, and highly performing user interface. Built fundamentally on Next.js, it offers real-time analytics in a sleek, dark-mode-first aesthetic.

## Dashboard Overview

![Splyt Main Dashboard](/portfolio/splyt2.webp)

Visualizing financial data requires clarity. The layout prioritizes the most critical metrics at the top, followed by interactive charts.

### The Analytics Engine

![Splyt Analytics View](/portfolio/splyt3.webp)

Powered by sophisticated charting libraries, Splyt allows users to drill down into their spending habits, investment growth, and overall financial health.

![Splyt Transaction History](/portfolio/splyt4.webp)

## Design System

The Splyt design system uses a strict monochromatic scale accented by vibrant, semantic colors to indicate financial status (green for growth, red for loss).

![Splyt Design System Elements](/portfolio/splyt5.webp)

![Splyt Mobile Interface](/portfolio/splyt6.webp)

All components are strictly typed with TypeScript, ensuring that the heavy data load doesn't compromise application stability.

# Nova : Portfolio d'Agence Premium 🌌

Nova n'est pas juste un modèle ; c'est une déclaration d'intention. Conçu pour les agences créatives repoussant les frontières de l'espace numérique, Nova offre une esthétique brutaliste audacieuse combinée à des animations de défilement ultra-fluides propulsées par GSAP et Next.js.

## L'Élégance Brutaliste

![Nova - Hero Page d'Accueil](/portfolio/nova2.webp)

Nova exploite l'espace négatif, les typographies brutalistes et les mises en page asymétriques pour capter l'attention immédiatement.

## Excellence du Dark Mode

![Nova - Vitrine de Projets](/portfolio/nova3.webp)

Chaque pixel est optimisé pour des dark modes profonds et immersifs, mettant en valeur le contraste saisissant entre la typographie, les images et le vide.

![Nova - Détails de l'Agence](/portfolio/nova4.webp)

### Déclencheurs de Défilement Avancés

En intégrant profondément `GSAP ScrollTrigger` dans les composants React, le portail offre un flux parallaxe qui s'apparente davantage à des applications natives qu'à des documents web standards.

![Nova - Mise en Page des Services](/portfolio/nova5.webp)

**Nova** représente le summum du design frontend moderne pour les agences.

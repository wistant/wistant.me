# Swarm AI Orchestration

This project demonstrates the power of multi-agent systems in content creation. By piping specialized agents, we achieve a depth and structure that single-prompt systems often struggle with. Powered by <TechBadge name="TypeScript" /> and <TechBadge name="Linux" /> environments.

## The Vision

The goal was to create a research-assistant that doesn't just write, but plans and verifies. We leveraged **Pydantic v2** to ensure that the "Planner" agent doesn't just hallucinate a structure, but follows a strict JSON schema that the "Writer" can reliably consume.

## Multi-Agent Pipeline

The system is standardized on **Llama 3.3 70B via Groq** for superior reasoning.

- **Planner Agent**: Generates a structured 5-section outline using Pydantic-validated schema.
- **Researcher Agent**: Performs deep-dive research for each planned section.
- **Writer Agent**: Crafts the final 1000+ word Markdown article.
- **PDF Generator**: Sanitizes Markdown and exports professional reports using **FPDF2**.

## Technical Challenges

### Agent Synchronization
Designed a linear, sequential pipeline where each agent's validated output is passed as the structured input to the next, eliminating race conditions.

### Schema Enforcement
Leveraged Pydantic v2's model validation to create strict contracts for every LLM response, ensuring the pipeline never consumes malformed data.

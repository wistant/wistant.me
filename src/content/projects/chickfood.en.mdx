# Chickfood: The Modern Fast Food Experience 🍔

Creating an online presence for a fast-food brand requires striking the perfect balance between **appetizing visuals** and **instant performance**. Chickfood is a modern frontend built with Next.js designed to make users hungry and drive fast conversions.

## Visual Appetizer

![Chickfood App Concept](/portfolio/chickfood2.webp)

## Smooth Interactions with Framer Motion

We wanted the UI to feel "bouncy" and alive, just like the energy of a bustling modern diner. By using `framer-motion`, we animated elements as they enter the viewport.

![Chickfood Mobile View](/portfolio/chickfood3.webp)

### Key Features
- **Vibrant Aesthetic**: Bold colors that stimulate appetite.
- **Scroll Animations**: Elements pop into place seamlessly.
- **Responsive Geometry**: Perfectly scales from mobile phones to massive desktop monitors.

![Chickfood Interactive Menu](/portfolio/chickfood4.webp)

## The Tech Stack
Built fundamentally on **Next.js** for SSR performance and SEO, paired with **Tailwind CSS** for rapid styling, and **React** component architecture to keep the menu system highly maintainable.
